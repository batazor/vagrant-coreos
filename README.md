# vagrant-coreos-kubernetes

### Getting start

```
vagrant up
```

### Setting

For settings use file `.env`

| **NAME**              | **Default**   | **Description**                      |
|-----------------------|---------------|--------------------------------------|
| NUM_INSTANCES         | 1             | count machine                        |
|-----------------------|---------------|--------------------------------------|
| INSTANCE_NAME_PREFIX  | core          |                                      |
| UPDATE_CHANNEL        | stable        |                                      |
| IMAGE_VERSION         | current       |                                      |
|-----------------------|---------------|--------------------------------------|
| VM_GUI                | false         |                                      |
| VM_MEMORY             | 1024          |                                      |
| VM_CPUS               | 1             |                                      |
| VB_CPUEXECUTIONCAP    | 100           |                                      |
|-----------------------|---------------|--------------------------------------|
